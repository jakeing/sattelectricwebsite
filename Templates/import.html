Required Excel Format

Your Excel file should have the following columns:

  • customer_work_order_number - External reference number from customer
  • satt_job_number - Unique internal job number (required)
  • description - Brief description of the work (required)
  • status - Status of the work order (Open, Complete, Closed)
  • owner - Customer or individual who owns this work order
  • estimated_hours - Estimated time to complete the work
  • priority - Priority level (Low, Medium, High)
  • location - Location where work will be performed
  • scheduled_date - Date work is scheduled (YYYY-MM-DD format)
Example Format:
customer_work_order_number satt_job_number description status owner
CWO-12345 SATT-001 Install new equipment Open ACME Corp
CWO-12346 SATT-002 Maintenance service Open XYZ Inc
Maximum file size: 10MB

Help & Tips
  • Unique Job Numbers: Ensure each SATT job number is unique. Duplicate job numbers will not be imported.
  • Required Fields: At minimum, include SATT_job_number and description for each work order.
  • Date Format: Use YYYY-MM-DD format for scheduled dates. Example: 2025-05-07.
