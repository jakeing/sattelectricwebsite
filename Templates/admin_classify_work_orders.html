Classification Rules

This tool will automatically classify your work orders using the following rules:

  • Billable: Work orders with a 6-digit customer work order number AND an SATT Job number
  • Non-Billable: Work orders with customer work order numbers of "0", "00", "000", or "0000"
  • All other work orders will be classified as Billable by default

Note: Existing "Contract/Project" classifications will not be changed.

Warning

This action will update the classification field for all work orders in the system.

This operation cannot be undone automatically. Please ensure you have a database backup before proceeding.
